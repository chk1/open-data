Was ist Open Data

"Offene Daten sind sämtliche Datenbestände, die im Interesse der Allgemeinheit der Gesellschaft ohne jedwede Einschränkung zur freien Nutzung, zur Weiterverbreitung und zur freien Weiterverwendung frei zugänglich gemacht werden."

1. "5 ★ OPEN DATA"

http://5stardata.info

Tim Berners-Lee schlägt 5 Sterne-Schema vor:

  1. ★ Verfügbar im Web unter offener Lizenz
  2. ★★ Strukturiert (Excel statt gescanntes PNG)
  3. ★★★ Nicht-proprietäres Format (CSV)
  4. ★★★★ Eindeutige URI
    (RDF / Atom feed, Semantisches Web)
  5. ★★★★★ Verlinkt & mit Kontext

2. "Global Open Data Index"

http://index.okfn.org/
